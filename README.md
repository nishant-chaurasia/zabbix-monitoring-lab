# Zabbix Monitoring Lab

This project demonstrates the setup of a monitoring infrastructure using Zabbix.

The lab environment was created using a virtualized setup with an Ubuntu server acting as the Zabbix monitoring server and two Windows machines configured with Zabbix agents.

Project Objectives:

- Install and configure Zabbix server
- Configure MySQL database for monitoring data
- Deploy Zabbix agents on Windows hosts
- Monitor system metrics such as CPU, memory, and network usage
- Visualize monitoring data through the Zabbix dashboard

Environment:

Hypervisor: Hyper-V  
Server OS: Ubuntu  
Monitoring Tool: Zabbix  
Database: MySQL  
Monitored Systems: Windows machines with Zabbix agent
